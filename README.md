# Life-Savers

Sistema de Agendamento Médico

O Life Savers é um sistema de agendamento médico desenvolvido para otimizar o gerenciamento de consultas em clínicas, hospitais e consultórios, reduzindo filas presenciais e melhorando a organização dos atendimentos.

O sistema permite que pacientes realizem agendamentos online, enquanto médicos e administradores conseguem gerenciar horários, consultas e disponibilidade de forma simples e eficiente.

---------- Objetivo do Projeto ----------

O principal objetivo do sistema é:

- Reduzir filas de atendimento;
- Evitar conflitos de horários;
- Facilitar o gerenciamento de consultas;
- Melhorar a experiência do paciente;
- Automatizar processos de agendamento médico.
